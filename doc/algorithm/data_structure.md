## 数组（连续存储）

- 数组存放的所有元素在内存中是相连的（一块内存空间）
- 添加新元素的速度会很慢（因为元素相连，增加元素的时候如果没有了空间需要重新申请内存空间全部转移元素）
- 随机读取元素时，数组的效率很好，因为可以很快的找到索引位置
