# 常用容器

以 JAVA 为例，说明常用容器的实现方式

| 容器          | 数据结构           | 线程安全 | 容量 |
| :------------ | :----------------- | :------- | :--- |
| ArrayList     | 数组               | 否       |      |
| Vector        | 数组               | 是       |      |
| Stack         | Vector             | 是       |      |
| LinkedList    | 双向链表           | 否       |      |
| HashSet       | HashMap            | 否       |      |
| LinkedHashSet | LinkedHashMap      | 否       |      |
| TreeSet       | 带全排序的集合容器 | 否       |      |
